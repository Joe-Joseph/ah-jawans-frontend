# Setting up Dev

## Clone the Repository to your local machine <br/>

```
git clone https://github.com/andela/ah-jawans-frontend.git
```

## Install dependencies <br/>

```
npm install
```

## Starting development server <br/>

```
npm run dev
```

## Run Tests <br/>

```
npm run test
```

## Contribute

